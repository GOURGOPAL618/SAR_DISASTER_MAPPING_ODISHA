## ⚡ MISSION OVERVIEW

This repository constitutes the **complete operational codebase and geospatial intelligence framework** for spaceborne Synthetic Aperture Radar (SAR) inundation mapping across the **coastal Odisha disaster zone**, developed and maintained by **Gouragopal Mohapatra** under Mission ID `OM-ODISHA-S1-UNET-02`.

The system intercepts **Sentinel-1 C-Band SAR backscatter anomalies** across hydrologically volatile terrain — where conventional optical sensors (Sentinel-2, Landsat) are rendered entirely non-operational due to persistent monsoon cloud cover — and applies a custom **U-Net deep learning segmentation pipeline** to extract high-confidence flood inundation footprints at operationally actionable precision.

> **Operational Context:** Coastal Odisha represents one of the most hydrologically volatile landscapes on the Indian subcontinent, characterized by expansive river deltas, low-lying prograding coastlines, and direct cyclonic exposure to the Bay of Bengal. This system addresses the critical information blackout created when cloud cover eliminates all optical Earth Observation capacity during peak disaster cycles.

## 🔬 BACKSCATTER CLASSIFICATION TAXONOMY

The U-Net framework isolates distinct spatial categories by exploiting **localized backscatter dynamics** via multi-temporal differential feature matrices:

| Feature Class | Backscatter Profile (σ⁰) | Physical Scattering Mechanism | Classification Output |
|---|---|---|---|
| **Permanent Deep Water** | Extreme Low (≤ −22 dB) | Specular reflection — null echo return to sensor | Filtered Base Layer (Masked) |
| **Active Flood Inundation** | Significant Drop (Δσ⁰ ≤ −6 dB) | Surface replacement by standing water; specular transition | **Value 1 · Neon Blue Target** |
| **Submerged Agriculture** | Moderate Drop + Double-Bounce | Emergent vegetation stalk interaction with water surface | **Value 1 · Classified Inundation Zone** |
| **Dry Mainland Terrain** | Stable High Scale (−14 to −8 dB) | Diffuse rough-surface scattering — soil and built surfaces | Value 0 · Transparent Background |
| **Dense Coastal Mangroves** | Extreme High (−6 to −3 dB) | Volume scattering inside complex branch canopy structure | Value 0 · Preserved Structural Boundary |

## ⚙️ PIPELINE EXECUTION ENGINE (`src/`)

### `src/dataset.py` — SAR Data Ingestion & Stream Generation
Manages the ingestion of **co-registered TIF scene containers**, applies **division-by-zero mitigation constraints** during logarithmic ratio compilation, and constructs the real-time tensor pipeline array stream formatted to the locked 256×256×3 token dimension standard.

### `src/model.py` — U-Net Architecture & Loss Computation
Implements the complete mathematical layer stack of the structural segmentation network, incorporating the **customized Sorenson-Dice loss matrix** engineered to resolve extreme class imbalances inherent in flood-to-land pixel ratio distributions.

### `src/inference.py` — Spatial Sliding-Window Inference Engine
Executes a **non-overlapping spatial sliding-window matrix** across the full-resolution satellite scene frame, re-assembling discrete tile-level outputs into a contiguous, georeferenced prediction array with full metadata alignment.

## 🗺️ GIS CARTOGRAPHIC OUTPUT PROTOCOLS (QGIS)

Post-processing transforms raw probability matrices into **operational vector intelligence maps** via strict cartographic protocols:

**Affine Geotransform Coordinate Rebuilding:**
```
X_geo = A · X_pixel + B · Y_pixel + C
Y_geo = D · X_pixel + E · Y_pixel + F
```
Every predicted pixel index is mapped back to absolute geographic coordinates via this affine transformation, locking the output array to **WGS 84 / EPSG:4326** with zero spatial offset against Google Satellite and OpenStreetMap base layers.

**Discrete Cartographic Visualization Schema:**
- `Value = 0` → **Absolute Transparency (0% Opacity)** — Roads, urban settlements, and unaffected terrain remain fully visible on base imagery.
- `Value = 1` → **Bright Cyan / Neon Blue (100% Opacity)** — Active inundation zones, river boundaries, and submerged agricultural tracts rendered with maximum visibility classification.

> Linear color interpolation is strictly prohibited. Only discrete binary classification rendering is operationally authorized.
